# CZSC ShareData 说明

本目录存放投研共享数据相关脚本与说明文件，主要用于离线数据准备与读取。

## 目录作用

- `marketData.py`：市场行情共享数据处理脚本
- `tushareData.py`：Tushare 共享数据处理脚本
- `parquet_info.txt`：parquet 数据说明与补充信息

## 使用定位

- 该目录偏“数据准备层”，常用于数据落地、预处理与共享；
- 上层通常通过 `connectors/research.py` 或其他读取接口消费这些数据。

## 使用建议

- 保持数据目录结构稳定，避免上层读取路径失效；
- 增量更新脚本建议记录更新时间与数据范围；
- 大体量 parquet 文件建议统一压缩与命名规范，方便检索与归档。
