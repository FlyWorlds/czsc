# CZSC Signals 说明

本目录是 CZSC 的信号函数库，提供多类别、可组合的技术信号实现。

## 目录作用

- `bar.py`：K线行为类信号
- `tas.py`：技术指标类信号（MA / MACD / KDJ / RSI / BOLL 等）
- `cxt.py`：缠论语义上下文信号
- `jcc.py`：K线形态（经典形态）信号
- `vol.py`：成交量相关信号
- `pos.py`：持仓与仓位管理相关信号
- `ang.py`：扩展指标类信号
- `coo.py` / `byi.py` / `zdy.py`：其他类别与自定义信号
- `__init__.py`：统一导出常用信号函数

## 规范特点

- 信号函数通常遵循“类别前缀 + 版本后缀”命名，例如 `xxx_xxx_V230101`；
- 通过版本后缀可并存多代实现，便于回测复现实验。

## 使用建议

- 不同信号之间存在相关性，组合前建议做去冗余分析；
- 对关键信号版本做冻结，避免策略结果因函数升级漂移；
- 新增信号建议同步提供最小示例与离线回测验证。
