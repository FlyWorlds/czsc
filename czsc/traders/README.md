# CZSC Traders 说明

本目录是交易与回测执行层，负责把信号、仓位与绩效评估串成可运行流程。

## 目录作用

- `base.py`：交易员核心对象与信号驱动执行框架（`CzscTrader` 等）
- `dummy.py`：简化回测执行器（`DummyBacktest`）
- `performance.py`：交易对与持仓绩效评估工具
- `weight_backtest.py`：权重回测与多策略组合评估
- `sig_parse.py`：信号配置解析与频率提取
- `optimize.py`：开仓/平仓参数优化工具
- `rwc.py`：`RedisWeightsClient` 等权重管理能力
- `__init__.py`：统一导出常用交易入口

## 核心能力

- **信号到交易**：将信号序列映射为仓位变化与交易动作
- **回测评估**：支持收益、回撤、交易对等多维绩效统计
- **策略优化**：支持参数组合优化与结果比较

## 使用建议

- 先在 `dummy` 或离线回测环境验证，再对接实盘连接器；
- 信号与交易规则应分层管理，降低调参耦合；
- 评估时建议同时查看收益、回撤、换手、交易成本敏感性。
