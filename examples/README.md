# CZSC Examples 说明

本目录提供示例脚本，覆盖策略开发、回测、实时运行、可视化与离线测试。

## 目录作用

- 根目录示例：快速上手脚本（策略回测、信号验证、QMT/GM 示例等）
- `develop/`：开发验证类脚本（功能实验、流程验证）
- `signals_dev/`：信号开发与调试示例
- `streamlit_pages/`：可视化页面与交互式研究示例
- `animotion/`：动画回放与可视化模板示例
- `test_offline/`：离线测试脚本（示例级）
- `dropit/`：临时或实验性脚本

## 使用建议

- 示例脚本默认可能依赖本地数据路径或第三方环境，运行前先检查配置；
- 建议复制为自己的实验脚本后再改，避免污染原始示例；
- 生产环境逻辑建议迁移到正式模块，不直接依赖示例脚本。
